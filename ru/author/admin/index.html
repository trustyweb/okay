VPRD на основе Linux Mint 17.1 и NX NoMachine

Самый простой способ, засекайте 10 минут.

подробнее
Это простой и быстрый способ настройки VPRD на основе популярной Open Source операционной системы Linux Mint и бесплатного ПО для удаленного доступа от заслуживающей доверия компании NX Nomachine. Протокол Nomachine является на данный момент лучшим решением для мультимедиа «без границ» — вы можете без проблем смотреть видео и играть в игры на удаленном компьютере. Даже с планшета и смартфона.

Нам подойдет любой VPS-хостер, позволяющий загрузить свой ISO образ и работать с VPS через встроенную в панель управления VNC-консоль. Для примера возьмем vultr.com – это хостер отвечающий всем нашим требованиям. Из приятного, у Vultr вы можете выбрать месторасположение вашего VPRD из большого списка локаций по всему миру. Цены за аренду дешевые, а оплата возможна в том числе и через bitcoin.

  • Переходим на http://vultr.com/ и регистрируемся. *приятный бонус – после пополнения баланса вам на счет добавят лишних 5$

  • Пополняем баланс любым удобным способом – минимум 5$. *не забываем подтвердить учетную запись по ссылке, присланной на email.

  • После пополнения баланса, загружаем установочный iso образ OS Linux Mint. Для этого заходим в личном кабинете в раздел «Upload Iso» и вставляем ссылку на дистрибутив в поле «Remote URL:» Ссылка прилагается ниже:
http://linuxfreedom.com/linuxmint/stable/17.1/linuxmint-17.1-mate-64bit.iso

Когда iso образ полностью загрузится, статус сменится на «Available»

  • Отлично. Идем в раздел «Deploy» и создаем наш первый приватный удаленный компьютер:
подробнее
• Location – это месторасположение удаленного компьютера. Выбирайте по вкусу, главное помните, что чем ближе сервер к вам, тем меньше обычно ping, а значит скорость работы с VPRD выше.

• Operating System – выбираем вкладку 64 bit и опцию Custom. Далее в Virtual ISO указываем загруженный ранее iso образ linuxmint-17.1-mate-64bit.iso

• Server Size – мощность нашего VPRD. Такие характеристики как количество процессорных ядер, оперативной памяти и емкости SSD диска. Для VPRD на основе desktop-дистрибутива, для комфортной работы рекомендуется от 2 CPU и 2gb оперативной памяти.

• Server Label – название приватного удаленного компьютера.

  • Только что созданный VPRD можно увидеть в разделе «My Servers» Ждем пока статус сервера сменится на «Running», потом нажимаем Manage

  • Все, теперь можно приступить к установке OS Linux Mint. Для этого нажимаем «View Console» – откроется окно с графическим экраном установки операционной системы.

Установка:

*язык, часовой пояс и раскладку клавиатуры выбирайте на свое усмотрение

  • Отлично, главное позади, осталось совсем немного. После установки нажмите «Restart Now» (*см. выше) и закройте окно VNC-консоли. Теперь необходимо «извлечь» установочный iso образ из виртуального компьютера:

  • Снова заходим в VNC-консоль, нажав на кнопку «View Console» в подразделе Server Actions. Ждем пока загрузится Linux, выбираем свою учетную запись и вводим к ней пароль:

  • Теперь необходимо скачать и установить на VPRD сервер удаленного доступа Nomachine. Открываем меню, запускаем Firefox и в открывшемся окне поиска ищем «NX Nomachine». Далее переходим на сайт Nomachine в раздел Downloads → NoMachine for Linux (Download) → NoMachine for Linux DEB amd64 → Download. Скачиваем и устанавливаем:

  • И последний штрих. Необходимо обновить систему. Menu → Terminal. В открывшемся окне терминала вводим:
sudo apt-get update && sudo apt-get dist-upgrade

и жмем Enter. Далее вводим пароль от учетной записи и жмем Enter (процесс ввода пароля никак не отображается в терминале – это нормально). После обновления списка пакетов вас попросят подтвердить действия → вводим заглавную Y и жмем Enter. Отлично, когда процесс обновления закончится, закройте терминал и перезагрузите OS. Закройте окно VNC-консоли, оно нам больше не понадобится.

 

На этом процесс установки и настройки VPRD закончился.

Скачайте клиент для удаленного доступа с сайта NoMachine под вашу OS (windows, mac, linux, ios или android). Все что потребуется для входа на удаленный компьютер – это его IP адрес и логин/пароль от учетной записи. IP адрес VPRD можно увидеть в панели управления хостингом, в разделе My Servers.
